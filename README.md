StyleMe is an app developed to help indecisive users to choose an outfit based on the specified event. This project was developed for the HackS'US V Hackathon 2026, conducted at Rajagiri School of Engineering and Technology, Kochi, Kerala, India.

Setup

1. Clone the repo
2. Create a .env file in the root directory
3. Add your Groq API key

Example:

VITE_GROQ_API_KEY=your_api_key_here

4. Run the project

npm install
npm run dev
